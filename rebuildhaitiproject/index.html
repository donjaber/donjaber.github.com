  • What

    The "Rebuild Haiti Project" is a one page index of online information pertinent to Haiti.
    The index packs a lot of information into a simple and organized interface allowing you to drill down the subjects which matter to you.
    The included links of externally maintained websites are carefully selected to be as fresh, relevant, authoritative, complete, as possible.

    Why

    The thought behind the page was to provide some sort of "Dashboard" of Haiti which could be used to better understand the country. An educated citizen is of better service to the country. An educated tourist is more attached to the country.

    How

    There are several areas to this page.

    1. The Background index provides links to timeless information about Haiti.
    2. The Info index provides links to current information about Haiti.
    3. The Multimedia index provides links to different media allowing you to more deeply enjoy and learn what the country has to offer.
    4. The Page Administration section provides a way to interact with the current page.
    5. The right panel section provides a quick view of current news and what others are currently saying about Haiti.
    6. The Search section provides a way to search the .ht (Haiti's own network) top level domain for more information.

    Welcome!

    Enjoy learning more about the country you love, will someday visit or invest in!
